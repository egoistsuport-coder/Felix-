# 🧪 ТЕСТИРОВАНИЕ ИНЛАЙН-КНОПОК

## После деплоя проверить:

### ✅ Команда /start
```
Отправить: /start
```
**Ожидается:**
- Приветственное сообщение
- 4 инлайн-кнопки:
  - 📚 Помощь
  - 👤 Профиль
  - 🤖 AI Команды
  - 📱 Открыть Mini App

---

### ✅ Кнопка "Помощь"
```
Нажать: 📚 Помощь
```
**Ожидается:**
- Сообщение "Felix Bot v5.0"
- 3 кнопки:
  - 🤖 AI Команды
  - 👤 Профиль / 📊 Статистика
  - 📱 Mini App

---

### ✅ Кнопка "AI Команды"
```
Нажать: 🤖 AI Команды
```
**Ожидается:**
- Сообщение "AI Команды"
- 8 кнопок команд:
  - 💬 Задать вопрос
  - 📝 Краткое содержание
  - 🔍 Анализ
  - ✨ Генерация
  - 🌐 Перевод
  - ✍️ Улучшить
  - 💡 Идеи
  - 📖 Объяснить
- Кнопка "« Назад"

---

### ✅ Кнопка "Профиль"
```
Нажать: 👤 Профиль
```
**Ожидается:**
- Профиль обучения
- Прогресс, сообщения, стиль
- Интересы (если есть)
- Кнопка "« Назад"

---

### ✅ Кнопка "Mini App"
```
Нажать: 📱 Открыть Mini App
```
**Ожидается:**
- Открывается Mini App в Telegram
- URL: https://felix-black.vercel.app/miniapp/
- 4 вкладки: Profile, Commands, Group, Settings

---

### ✅ Навигация
```
Проверить все кнопки "Назад"
```
**Ожидается:**
- Из AI команд → Помощь
- Из Профиля → Помощь
- Из Статистики → Помощь
- Все переходы работают

---

### ✅ AI Команды через кнопки
```
Нажать на любую AI команду
```
**Ожидается:**
- Показывается инструкция по использованию
- Пример команды
- Кнопка "« Назад"

---

### ✅ AI Команды через текст
```
Отправить: /ask привет как дела
```
**Ожидается:**
- AI ответ на русском
- Адаптирован под стиль пользователя

---

## 🎯 КРИТИЧЕСКИЕ ПРОВЕРКИ:

1. ✅ Все кнопки кликабельны
2. ✅ Callback_query обрабатывается
3. ✅ Mini App открывается
4. ✅ Навигация работает
5. ✅ AI команды отвечают
6. ✅ Профиль показывает данные
7. ✅ Модерация работает в группах

---

## 🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

### Кнопки не реагируют:
- Проверить логи Vercel
- Убедиться что callback_query обрабатывается

### Mini App не открывается:
- Проверить URL: https://felix-black.vercel.app/miniapp/
- Убедиться что файл miniapp/index.html существует

### AI не отвечает:
- Проверить GROQ_API_KEY
- Проверить лимиты API
