# 🎨 Felix Mini App - Полная спецификация дизайна v5.0

## 🎯 КОНЦЕПЦИЯ ДИЗАЙНА:

### Стиль: Modern Glassmorphism + Gradient
- Прозрачные карточки с размытием (backdrop-filter)
- Градиентные акценты (золото → красный)
- Плавные анимации и переходы
- Темная тема с яркими акцентами
- Адаптивный дизайн для всех устройств

### Цветовая палитра:
```css
--primary: #667eea (Синий)
--secondary: #764ba2 (Фиолетовый)
--accent: #ffd700 (Золото)

--danger: #f44336 (Красный)
--warning: #ff9800 (Оранжевый)
--info: #2196f3 (Голубой)
```

## 🎨 ЭФФЕКТЫ:

### Glassmorphism:
```css
background: rgba(255, 255, 255, 0.1);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.2);
```

### Gradient Text:
```css
background: linear-gradient(135deg, #fff, #ffd700);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```

### Glow Effect:
```css
box-shadow: 0 8px 32px rgba(255, 215, 0, 0.4);
text-shadow: 0 2px 10px rgba(255, 215, 0, 0.4);
```

### Ripple Effect:
```css
/* При клике создается расходящийся круг */
transform: translate(-50%, -50%);
width: 0 → 300px;
height: 0 → 300px;
opacity: 1 → 0;
```
