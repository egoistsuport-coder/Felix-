## 🚀 Что получаем

### /translate PRO
```
Базовое: /translate en Текст
Опции:
  :formal - формальный стиль
  :casual - неформальный
  :technical - технический
  :literary - литературный
  :verify - с обратным переводом
  :context=X - с контекстом

Примеры:
  /translate:en Привет!
  /translate:en:formal Здравствуйте
  /translate:en:verify:context=business Договор
```

### /improve PRO
```
Базовое: /improve Текст
Опции:
  :grammar - только грамматика
  :style - только стиль
  :clarity - ясность
  :concise - сократить
  :detailed - расширить
  :professional - профессиональный
  :explain - с объяснениями
  :light/medium/heavy - уровень изменений
  :audience=kids/general/experts - аудитория

Примеры:
  /improve Я пошол в магазин
  /improve:grammar:explain Текст
  /improve:concise:audience=kids Текст
```

### /brainstorm PRO
```
Базовое: /brainstorm Тема
Опции:
  :count=N - количество идей
  :categorize - по категориям
  :rate - с оценкой
  :detailed - подробно
  :creative=low/medium/high/wild - креативность

Примеры:
  /brainstorm Идеи для YouTube
  /brainstorm:count=20:categorize Бизнес
  /brainstorm:rate:detailed:creative=wild Инновации
```

### /explain PRO
```
Базовое: /explain Тема
Опции:
  :simple - для детей
  :medium - для обычных людей
  :advanced - для знающих
  :expert - для экспертов
  :examples - с примерами
  :analogies - с аналогиями
  :steps - пошагово
  :visual - с диаграммами

Примеры:
  /explain Квантовая физика
  /explain:simple:examples Блокчейн
  /explain:analogies:steps:visual Нейросети
```

### /stats PRO
```
Базовое: /stats
Опции:
  :detailed - подробная статистика
  :chart - с графиками
  :commands - только команды
  :time - только время

Примеры:
  /stats
  /stats:detailed:chart
  /stats:commands
```
