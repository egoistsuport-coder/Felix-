## 🚀 Пошаговый план

### Шаг 1: Supabase (1 час)
1. Создать проект на supabase.com
2. Применить `database/v4-schema.sql`
3. Настроить Storage buckets
4. Получить credentials

### Шаг 2: Environment Variables (10 мин)
1. Обновить `.env.local`
2. Добавить в Vercel Environment Variables

### Шаг 3: Интеграция БД (2 часа)
1. Импортировать `lib/db.js` в `api/webhook.js`
2. Заменить `Map` на `db` функции
3. Обновить команды для работы с БД

### Шаг 4: Learning System (2 часа)
1. Добавить вызовы Learning API
2. Добавить уведомления о достижениях
3. Добавить команды: /level, /achievements, /tasks

### Шаг 5: Тестирование (1 час)
1. Тестировать БД
2. Тестировать Learning System
3. Тестировать уведомления

### Шаг 6: Деплой (30 мин)
1. Коммит и push
2. Проверить деплой на Vercel
3. Протестировать в production
