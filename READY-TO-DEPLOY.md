## 🔧 Troubleshooting

### Ошибка: Module not found
```bash
# Проверьте что файл называется точно:
api/webhook.js

# И содержит правильный код из webhook-v4-simple.js
```

### Ошибка: GROQ_API_KEY not found
```bash
# Проверьте в Vercel → Settings → Environment Variables
# Должна быть переменная GROQ_API_KEY
```

### Бот не отвечает
```bash
# 1. Проверьте логи в Vercel
# 2. Проверьте что deployment успешный
# 3. Проверьте webhook URL в Telegram
```

### Голосовые не работают
```bash
# Проверьте GROQ_API_KEY
# Проверьте лимиты Groq API
```
