## 🧪 ТЕСТИРОВАНИЕ:

### 1. Проверка API:
```
https://felix-black.vercel.app/api/admin?action=isAdmin&userId=8264612178
```
Ответ: `{"isAdmin": true}`

### 2. Открыть админ-панель:
```
https://felix-black.vercel.app/miniapp/admin.html
```
- Если ID = 8264612178 → админ-панель
- Если другой ID → "Доступ запрещен"

### 3. Подать заявку:
1. Открыть Mini App: https://felix-black.vercel.app/miniapp/
2. Вкладка "🤝 Партнеры"
3. Нажать "📝 Подать заявку"
4. Заполнить форму
5. Отправить

### 4. Проверить заявку в админке:
1. Открыть админ-панель
2. Вкладка "📝 Заявки"
3. Увидеть новую заявку

### 5. Одобрить заявку:
1. Нажать "✅ Одобрить"
2. Выбрать логотип
3. Отметить Featured (опционально)
4. Одобрить
5. Проверить вкладку "🤝 Партнеры" → партнер добавлен

### 6. Добавить курс:
1. Вкладка "🎓 Курсы"
2. Нажать "➕ Добавить курс"
3. Заполнить форму
4. Добавить
5. Открыть Mini App → курс виден
