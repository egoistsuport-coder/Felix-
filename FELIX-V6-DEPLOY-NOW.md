## 🔄 Шаги для деплоя

### Шаг 1: Заменить webhook.js
```bash
# Сделать бэкап старого файла
mv api/webhook.js api/webhook-v5-backup.js

# Переименовать новый файл
mv api/webhook-v6.js api/webhook.js
```

Или вручную:
1. Удалить `api/webhook.js`
2. Переименовать `api/webhook-v6.js` → `api/webhook.js`

### Шаг 2: Проверить изменения
```bash
git status
```

Должны быть:
- `modified: miniapp/index.html`
- `modified: api/webhook.js`
- `new file: api/learning.js`
- `new file: api/voice.js`

### Шаг 3: Коммит
```bash
git add miniapp/index.html api/webhook.js api/learning.js api/voice.js
git commit -m "Felix v6.0: Complete - Voice control, learning system, bot integration"
```

### Шаг 4: Push через GitHub Desktop
1. Открыть GitHub Desktop
2. Увидите коммит "Felix v6.0: Complete..."
3. Нажать "Push origin"
4. Vercel автоматически задеплоит
