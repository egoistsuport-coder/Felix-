### 2. Cache Service (`lib/cache.js`)
Кэширование AI ответов для оптимизации и экономии API запросов.

**Возможности:**
- ✅ Кэширование результатов AI
- ✅ Настраиваемый TTL для разных операций
- ✅ Автоматическая очистка устаревших записей
- ✅ Статистика кэша
- ✅ In-memory хранилище (быстро)

**TTL (время жизни):**
```javascript
{
  summary: 5 минут,
  analyze: 10 минут,
  generate: 5 минут,
  organize: 10 минут,
  voice: 30 минут,
  text: 5 минут
}
```

**Преимущества:**
- ⚡ Быстрые ответы на повторяющиеся запросы
- 💰 Экономия Groq API лимитов
- 📈 Улучшенная производительность

### 3. Rate Limiting Service (`lib/ratelimit.js`)
Защита от злоупотреблений и спама.

**Возможности:**
- ✅ Лимиты по командам
- ✅ Глобальный лимит на пользователя
- ✅ Настраиваемые окна времени
- ✅ Автоматический сброс лимитов
- ✅ Статистика использования

**Лимиты:**
```javascript
{
  '/summary': 5 запросов/минуту,
  '/analyze': 10 запросов/минуту,
  '/generate': 5 запросов/минуту,
  '/organize': 10 запросов/минуту,
  'voice': 10 запросов/минуту,
  'text': 30 запросов/минуту,
  'global': 50 запросов/минуту
}
```

**Ответ при превышении:**
```javascript
{
  allowed: false,
  remaining: 0,
  resetAt: 1709308800000,
  retryAfter: 45 // секунд
}
```

REST API для получения статистики.

## 📊 Интеграция в webhook.js

### Добавить импорты:
```javascript
import analytics from '../lib/analytics.js';
import cache from '../lib/cache.js';
import rateLimiter from '../lib/ratelimit.js';
```

### Использование в командах:

#### 1. Rate Limiting
```javascript
// Проверка лимита перед обработкой
const limit = rateLimiter.checkLimit(userId, '/summary');
if (!limit.allowed) {
  await sendMessage(chatId, 
    `⏳ Превышен лимит запросов. Попробуйте через ${limit.retryAfter} секунд.`
  );
  return res.status(200).json({ ok: true });
}
```

#### 2. Кэширование
```javascript
// Проверка кэша перед AI запросом
const cached = cache.get('analyze', textToAnalyze);
if (cached) {
  await sendMessage(chatId, `🔍 <b>Анализ текста:</b>\n\n${cached}`);
  analytics.trackCommand(userId, '/analyze', 0);
  return res.status(200).json({ ok: true });
}

// Выполнение AI запроса
const analysis = await analyzeText(textToAnalyze);

// Сохранение в кэш
cache.set('analyze', textToAnalyze, analysis);
```

#### 3. Analytics
```javascript
// Трекинг команды
const startTime = Date.now();
const result = await createSummary(history.slice(-20));
const duration = Date.now() - startTime;

analytics.trackCommand(userId, '/summary', duration);

// Трекинг ошибки
try {
  // ... код ...
} catch (error) {
  analytics.trackError('summary_error', error.message);
  throw error;
}
```

## 🔧 Настройка

### Изменение лимитов
Отредактируйте `lib/ratelimit.js`:
```javascript
this.config = {
  commands: {
    '/summary': { limit: 10, window: 60000 }, // 10 вместо 5
    // ...
  }
};
```

### Изменение TTL кэша
Отредактируйте `lib/cache.js`:
```javascript
this.ttl = {
  summary: 10 * 60 * 1000, // 10 минут вместо 5
  // ...
};
```
