## 📋 ЧТО ДЕЛАТЬ В GITHUB DESKTOP:

### Шаг 1: Открыть GitHub Desktop
1. Запустить приложение
2. Убедиться что выбран репозиторий: **Асистент копирайтер**

### Шаг 2: Проверить изменения
В левой панели "Changes" должно быть:
```
✅ api/webhook.js (modified)
```

Если видите другие файлы (.md), можно их не коммитить сейчас.

### Шаг 3: Написать коммит

В нижней части слева есть два поля:

**Summary (обязательное поле):**
```
Felix v5.0 UI - Inline buttons + Mini App
```

**Description (необязательное, но рекомендуется):**
```
Добавлены инлайн-кнопки для всех команд и восстановлен Mini App

Новое:
- Инлайн-кнопки для /start, /help
- Меню AI команд с кнопками
- Кнопка открытия Mini App
- Callback_query обработчик
- Навигация между меню

Интерфейс стал максимально понятным!
```

### Шаг 4: Commit
Нажать синюю кнопку внизу слева:
```
[Commit to main]
```

### Шаг 5: Push
После коммита вверху появится кнопка:
```
[Push origin]
```
Нажать её.

### Шаг 6: Дождаться
- GitHub Desktop покажет прогресс
- Когда появится зеленая галочка - готово!
- Время: 10-30 секунд

## ❓ ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК:

### GitHub Desktop не видит изменения:
- Обновить: Repository → Refresh (Ctrl+R)
- Проверить что выбран правильный репозиторий

### Кнопка Push неактивна:
- Сначала нужно сделать Commit
- Проверить что Summary заполнен

### Push не работает:
- Проверить интернет-соединение
- Проверить что вы залогинены в GitHub Desktop

### Бот не отвечает после деплоя:
- Подождать 2-3 минуты (Vercel может деплоить дольше)
- Проверить логи Vercel: https://vercel.com/dashboard
- Проверить webhook: https://felix-black.vercel.app/api/webhook
