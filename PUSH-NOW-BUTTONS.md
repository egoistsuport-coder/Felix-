## 📋 ПОШАГОВАЯ ИНСТРУКЦИЯ:

### Шаг 1: Открыть GitHub Desktop
1. Запустить приложение GitHub Desktop
2. Убедиться что выбран репозиторий Felix

### Шаг 2: Проверить изменения
В левой панели должно быть:
- ✅ `api/webhook.js` (modified)

### Шаг 3: Написать коммит-сообщение

**Summary (заголовок):**
```
Felix v5.0 UI - Inline buttons + Mini App
```

**Description (описание):**
```
Добавлены инлайн-кнопки для всех команд и восстановлен Mini App

Новое:
- Инлайн-кнопки для /start, /help
- Меню AI команд с кнопками
- Кнопка открытия Mini App
- Callback_query обработчик
- Навигация между меню

Интерфейс стал максимально понятным!
```

### Шаг 4: Commit
- Нажать синюю кнопку **"Commit to main"** внизу слева

### Шаг 5: Push
- Нажать кнопку **"Push origin"** вверху
- Дождаться завершения (появится зеленая галочка)

### Шаг 6: Дождаться деплоя Vercel
- Vercel автоматически задеплоит за 1-2 минуты
- Можно проверить статус на https://vercel.com/dashboard

### Шаг 7: Тестировать бота!
1. Открыть Telegram
2. Найти @fel12x_bot
3. Отправить `/start`
4. Должны появиться 4 кнопки:
   - 📚 Помощь
   - 👤 Профиль
   - 🤖 AI Команды
   - 📱 Открыть Mini App
