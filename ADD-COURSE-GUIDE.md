# 📚 Как добавить курс в Академию

## Быстрая инструкция:

### Шаг 1: Открыть файл
```
miniapp/academy.json
```

### Шаг 2: Скопировать шаблон
```json
{
  "id": "course4",
  "title": "Название курса",
  "description": "Краткое описание",
  "icon": "🚀",
  "duration": "30 мин",
  "level": "beginner",
  "lessons": [
    {
      "id": "lesson1",
      "title": "Урок 1",
      "content": "Содержание урока",
      "duration": "10 мин"
    },
    {
      "id": "lesson2",
      "title": "Урок 2",
      "content": "Содержание урока",
      "duration": "20 мин"
    }
  ]
}
```

### Шаг 3: Заполнить данные

#### Курс:
- `id` - уникальный идентификатор (course4, course5...)
- `title` - название курса
- `description` - описание (1-2 строки)
- `icon` - emoji иконка (🤖 🧠 🛡️ 🚀 💡 ⚡)
- `duration` - общая длительность
- `level` - уровень сложности:
  - `beginner` - Начальный (зеленый)
  - `intermediate` - Средний (желтый)
  - `advanced` - Продвинутый (красный)

#### Уроки:
- `id` - уникальный идентификатор урока
- `title` - название урока
- `content` - содержание (описание)
- `duration` - длительность урока

### Шаг 4: Вставить в файл
Открыть `miniapp/academy.json` и добавить после последнего курса:

```json
{
  "courses": [
    {
      "id": "course1",
      "title": "Основы работы с Felix Bot",
      ...
    },
    {
      "id": "course2",
      "title": "Самообучение бота",
      ...
    },
    {
      "id": "course3",
      "title": "Модерация групп",
      ...
    },
    {
      "id": "course4",
      "title": "ВАШ КУРС",
      "description": "Ваше описание",
      "icon": "🚀",
      "duration": "30 мин",
      "level": "beginner",
      "lessons": [...]
    }
  ]
}
```

⚠️ Не забудьте запятую после предыдущего курса!

### Шаг 5: Сохранить и задеплоить
```bash
git add miniapp/academy.json
git commit -m "Add new course: НАЗВАНИЕ"
git push origin main
```

### Шаг 6: Проверить
1. Дождаться деплоя Vercel (1-2 минуты)
2. Открыть Mini App
3. Перейти на вкладку "🎓 Академия"
4. Увидеть новый курс!

## 🚫 Частые ошибки:

### Ошибка 1: Неправильный level
```json
// ❌ НЕПРАВИЛЬНО
"level": "easy"

// ✅ ПРАВИЛЬНО (только эти 3)
"level": "beginner"
"level": "intermediate"
"level": "advanced"
```

### Ошибка 2: Неуникальные ID уроков
```json
// ❌ НЕПРАВИЛЬНО - два lesson1
"lessons": [
  {
    "id": "lesson1",
    "title": "First"
  },
  {
    "id": "lesson1",
    "title": "Second"
  }
]

// ✅ ПРАВИЛЬНО
"lessons": [
  {
    "id": "lesson1",
    "title": "First"
  },
  {
    "id": "lesson2",
    "title": "Second"
  }
]
```

### Ошибка 3: Сумма длительности уроков
```json
// ⚠️ ВНИМАНИЕ
"duration": "30 мин",  // Общая длительность
"lessons": [
  { "duration": "10 мин" },  // 10
  { "duration": "15 мин" },  // 15
  { "duration": "8 мин" }    // 8
]
// Сумма: 33 мин ≠ 30 мин

// ✅ ПРАВИЛЬНО
"duration": "33 мин"
```

## 💡 Советы:

1. Начинайте с простых курсов (beginner)
2. Разбивайте на короткие уроки (5-15 мин)
3. Используйте понятные названия
4. Добавляйте практические примеры
5. Указывайте точную длительность

---

## 📊 Рекомендации по длительности:

- **Beginner:** 15-30 мин (2-3 урока)
- **Intermediate:** 30-45 мин (3-4 урока)
- **Advanced:** 45-90 мин (4-6 уроков)

---

## 🎯 Структура хорошего курса:

1. **Введение** (5-10 мин)
   - Что изучим
   - Зачем это нужно

2. **Основная часть** (60-70%)
   - Теория
   - Практика
   - Примеры

3. **Заключение** (10-15%)
   - Итоги
   - Следующие шаги
