## ✅ ЧТО ДОБАВЛЕНО:

### 1. Секция персонализации в настройках
- Загрузка своей аватарки (фото до 5MB)
- Выбор эмодзи аватара (5 вариантов)
- Выбор стиля общения (неформальный/формальный/смешанный)
- Выбор языка интерфейса (русский/английский/украинский)
- Выбор темы оформления (темная/светлая)

### 2. API для сохранения настроек
- Endpoint: `getUserSettings` - получение настроек
- Endpoint: `updateUserSettings` - обновление настроек
- Хранение в памяти (Map)

### 3. Автоматическое применение настроек
- Загрузка при старте Mini App
- Применение к аватару профиля
- Применение темы оформления
- Сохранение между сессиями

## 💾 СОХРАНЕНИЕ НАСТРОЕК:

### Процесс:
1. Пользователь изменяет настройки
2. Клик "💾 Сохранить настройки"
3. Отправка в API (POST /api/admin)
4. Сохранение в Map
5. Подтверждение пользователю
6. Haptic feedback

### Структура данных:
```javascript
{
    userId: 123456789,
    avatar: "data:image/jpeg;base64,...", // или null
    avatarType: "emoji", // или "image"
    avatarEmoji: "🧠",
    communicationStyle: "casual", // casual/formal/mixed
    language: "ru", // ru/en/uk
    theme: "dark", // dark/light
    notifications: true,
    createdAt: 1234567890,
    updatedAt: 1234567890
}
```

## 🎯 ИСПОЛЬЗОВАНИЕ:

### Для пользователя:
1. Открыть Mini App
2. Вкладка "⚙️ Настройки"
3. Секция "🎨 Персонализация"
4. Изменить настройки
5. Нажать "💾 Сохранить настройки"
6. Получить подтверждение

### Для разработчика:
```javascript
// Получение настроек
const response = await fetch(`/api/admin?action=getUserSettings&userId=${userId}`);
const data = await response.json();

// Обновление настроек
const response = await fetch('/api/admin', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        action: 'updateUserSettings',
        userId: userId,
        data: {
            avatar: "...",
            communicationStyle: "formal",
            language: "en",
            theme: "light"
        }
    })
});
```
