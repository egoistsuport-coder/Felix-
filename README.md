# 🤖 Felix Bot v4.0 - AI Ассистент

Умный Telegram бот с расширенными AI командами, голосовым вводом и красивым Mini App интерфейсом.

## ✨ Новые возможности v4.0

### 📊 Саммари диалогов
- Команда `/summary` для создания краткого саммари
- Анализ последних сообщений
- Выделение ключевых моментов

### 🔍 Анализ текста
- Команда `/analyze` для глубокого анализа
- Определение тональности (позитивная/негативная/нейтральная)
- Извлечение ключевых слов и тем
- Анализ структуры текста

### ✨ Генерация контента
- Команда `/generate` для создания контента
- Статьи, email, посты в соцсети
- Код и технические документы
- Идеи и концепции

### 🎤 Голосовой ввод
- Распознавание голосовых сообщений через Groq Whisper Large v3
- Поддержка русского языка
- Автоматическая обработка и ответ AI

### 💬 AI Диалоги
- Ответы на вопросы через LLaMA 3.3 70B
- Контекстная память (последние 10 сообщений)
- Естественное общение на русском языке

### 📝 Организация текста
- Команда `/organize` для структурирования текста
- Автоматическое создание списков и категорий
- Выделение ключевых моментов

### 📱 Mini App
- Красивый интерфейс с градиентным дизайном
- Демонстрация всех возможностей v4.0
- Примеры использования команд
- Технологический стек

## 📋 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню с кнопкой Mini App |
| `/summary` | Создать саммари диалога (минимум 5 сообщений) |
| `/analyze <текст>` | Анализ текста: тональность, ключевые слова, темы |
| `/generate <промпт>` | Генерация контента: статьи, email, посты, код |
| `/organize <текст>` | Структурировать и организовать текст |
| `/clear` | Очистить историю диалогов |
| Голосовое | Отправьте голосовое - оно будет распознано |
| Текст | Любое сообщение - получите AI ответ с контекстом |

## 🛠️ Технологии

- **Backend**: Node.js (Vercel Serverless)
- **AI**: Groq API
  - LLaMA 3.3 70B Versatile (текстовая генерация)
  - Whisper Large v3 (распознавание речи)
- **Database**: PostgreSQL (Supabase)
- **Hosting**: Vercel
- **Bot API**: Telegram Bot API
- **Frontend**: Vanilla JS + Telegram WebApp API

## 📊 База данных (опционально)

### Упрощенная версия
- Использует in-memory storage (Map)
- Не требует настройки БД
- Идеально для быстрого старта
- Все AI команды работают

### Полная версия
Для постоянного хранения данных можно подключить Supabase:

**Таблицы:**
- `users` - информация о пользователях
- `messages` - история сообщений
- `voice_messages` - голосовые сообщения
- `tags` - теги для организации
- `user_settings` - настройки пользователей

**Инструкция:** [database/SETUP-SUPABASE.md](database/SETUP-SUPABASE.md)

## 📝 Примеры использования

### Саммари диалога
```
Пользователь: /summary
Felix: 📊 Саммари диалога:

Основные темы:
• Обсуждение проекта Felix Bot
• Планирование новых функций
• Технические детали реализации

Ключевые решения:
• Использовать Groq API для AI
• Начать с упрощенной версии
• Добавить команды /analyze и /generate
```

### Анализ текста
```
Пользователь: /analyze Этот продукт превзошел все мои ожидания! Качество отличное, доставка быстрая.
Felix: 🔍 Анализ текста:

Тональность: Позитивная (95%)
Ключевые слова: продукт, качество, доставка, ожидания
Темы: отзыв о товаре, удовлетворенность покупкой
Эмоции: восторг, удовлетворение
```

### Генерация контента
```
Пользователь: /generate идеи для статьи о путешествиях по России
Felix: ✨ Сгенерированный контент:

10 идей для статьи о путешествиях по России:

1. Золотое кольцо: маршрут по древним городам
2. Байкал зимой: ледовые приключения
3. Камчатка: вулканы и дикая природа
4. Карелия: озера и северное сияние
5. Алтай: горы и духовные практики
...
```

### Организация текста
```
Пользователь: /organize Купить молоко яйца хлеб позвонить врачу записаться на стрижку
Felix: 📝 Организованный текст:

🛒 Покупки:
• Молоко
• Яйца
• Хлеб

📞 Звонки:
• Позвонить врачу

✂️ Записи:
• Записаться на стрижку
```

### Голосовое сообщение
```
Пользователь: [Голосовое: "Проанализируй мой последний отзыв"]
Felix: 🎤 Распознано: Проанализируй мой последний отзыв

💬 Ответ:
```
