## 🎓 ACADEMY.JSON - Управление курсами

### Структура:
```json
{
  "courses": [
    {
      "id": "course1",
      "title": "Название курса",
      "description": "Описание",
      "icon": "🤖",
      "duration": "15 мин",
      "level": "beginner",
      "lessons": [
        {
          "id": "lesson1",
          "title": "Урок 1",
          "content": "Содержание",
          "duration": "5 мин"
        }
      ]
    }
  ]
}
```

### Уровни сложности:
- `beginner` - Зеленый бейдж
- `intermediate` - Желтый бейдж
- `advanced` - Красный бейдж

### Как добавить курс:
1. Открыть `miniapp/academy.json`
2. Добавить новый объект в массив `courses`
3. Указать id, title, description, icon, duration, level
4. Добавить уроки в массив `lessons`
5. Сохранить файл
6. Курс появится автоматически!

## 🤝 PARTNERS.JSON - Управление партнерами

### Структура:
```json
{
  "partners": [
    {
      "id": "partner1",
      "name": "Название",
      "description": "Описание",
      "logo": "🎓",
      "url": "https://example.com",
      "category": "education",
      "featured": true
    }
  ]
}
```

### Категории:
- `education` - Образование
- `support` - Поддержка
- `services` - Сервисы

### Как добавить партнера:
1. Открыть `miniapp/partners.json`
2. Добавить новый объект в массив `partners`
3. Указать id, name, description, logo (emoji), url
4. Выбрать category
5. Установить `featured: true` для TOP бейджа
6. Сохранить файл
7. Партнер появится автоматически!

## 🎨 АНИМАЦИИ:

### Доступные анимации:
- `fadeIn` - Плавное появление
- `fadeInUp` - Появление снизу
- `slideDown` - Скольжение сверху
- `slideInLeft` - Скольжение слева
- `slideInRight` - Скольжение справа
- `scaleIn` - Увеличение
- `bounceIn` - Прыжок
- `pulse` - Пульсация
- `float` - Плавание
- `shimmer` - Мерцание
- `rotate` - Вращение
- `bounce` - Подпрыгивание
- `countUp` - Счетчик

### Timing:
- Все анимации используют `cubic-bezier(0.4, 0, 0.2, 1)` для плавности
- Длительность: 0.3s - 0.6s
- Задержка между карточками: 0.1s
