## 💡 Функциональность

### 1. Основные команды

#### /start
```
Функция: Приветствие и главное меню
Кнопки:
  - 📱 Открыть Mini App
  - 💬 Начать диалог
Описание: Показывает все возможности бота
```

#### /summary
```
Функция: Создание саммари диалога
Требования: Минимум 5 сообщений в истории
AI Model: LLaMA 3.3 70B
Temperature: 0.3
Output: Основные темы, решения, важные моменты
```

#### /analyze <текст>
```
Функция: Анализ текста
Требования: Минимум 10 слов
AI Model: LLaMA 3.3 70B
Temperature: 0.2
Output:
  - Тональность (позитивная/негативная/нейтральная)
  - Ключевые слова
  - Темы
  - Читаемость
  - Язык
```

#### /generate <промпт>
```
Функция: Генерация контента
Типы:
  - article: Статьи
  - email: Email письма
  - social: Посты в соцсети
  - code: Код с комментариями
  - ideas: Списки идей
AI Model: LLaMA 3.3 70B
Temperature: 0.7
Output: Сгенерированный контент
```

#### /organize <текст>
```
Функция: Структурирование текста
AI Model: LLaMA 3.3 70B
Temperature: 0.3
Output:
  - Заголовки
  - Списки
  - Категории
  - Ключевые моменты
```

#### /clear
```
Функция: Очистка истории диалогов
Действие: Удаляет все сообщения из памяти
Storage: In-memory Map
```

## 📊 Производительность

### Vercel Serverless
```
Cold Start: ~500ms
Warm Response: ~100-200ms
Timeout: 10 seconds (default)
Memory: 1024 MB (default)
Region: Auto (closest to user)
```

### AI API (Groq)
```
LLaMA 3.3 70B:
  - Latency: ~1-3 seconds
  - Tokens/sec: ~100-200
  - Max tokens: 2048

Whisper Large v3:
  - Latency: ~2-5 seconds
  - Accuracy: ~95%
  - Languages: 99+
```

### Storage
```
In-memory (Map):
  - Read: O(1)
  - Write: O(1)
  - Persistence: До перезапуска
  - Limit: RAM (1024 MB)

PostgreSQL (опционально):
  - Read: ~10-50ms
  - Write: ~20-100ms
  - Persistence: Постоянная
  - Limit: План Supabase
```

### Speed Insights
```
Метрики:
✅ FCP (First Contentful Paint)
✅ LCP (Largest Contentful Paint)
✅ FID (First Input Delay)
✅ CLS (Cumulative Layout Shift)
✅ TTFB (Time to First Byte)

Цели:
- FCP < 1.8s
- LCP < 2.5s
- FID < 100ms
- CLS < 0.1
- TTFB < 600ms
```
